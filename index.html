Smart Temperature Monitor

This project uses Arduino and DHT11 to monitor room temperature and humidity. The data is sent wirelessly to a display using an RF module.

Tools Used: Arduino, DHT11, RF Module, 16x2 LCD

Steps:

  • Connect the DHT11 sensor to Arduino
  • Transmit data via RF module
  • Receive and display on LCD
